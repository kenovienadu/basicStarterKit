Starter Kit

This starter kit is a basic kit to enable developers get started with basic front end projects. It comes with automated bundling and minification using Parcel, transpiling using Babel and linting using Eslint. Feel free to give a shoutout to the author (@kenovienadu) if you like this starter kit.
